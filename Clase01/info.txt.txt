Software : JDK Java Development kit (kit de desarrollo Java) 

Java :  Es una especificacion , no es lenguaje, el fundamento de java era tener un lenguaje libre que pueda
funcionar en cualquier sistema operativo y que no necesite compilarse.
C# es Privativo, a diferencia de Java.
Comunidad de java :  Java community process dictamina como es el lenguaje


JDK : Generalmente se descarga el 11 pero es privativo
OpenJDk : Tiene soporte de oracle y es totalmente libre y se puede elegir el soporte de software(Compiladores)
Ejemplo JDk amazon, JDk redHat , JDK Microsoft, JDk IBM.


Version LTS(Long term Support 8 años) Una version no LTS tiene soporte por 6 meses y las LTS por 8 años.
Version 8 , 11 y 17 Son versiones LTS.
Para ver la version de JAva tenemos que abrir el cmd y ponemos Java --version, en caso de querer instalar una nueva version 
hay que desistalar y instalar la nueva version
